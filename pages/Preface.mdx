# **PREFACE**

This document comprises all the necessary information required, including all the diagrams, the flowcharts, the technology stack used, the complete documentation, and all the prerequisites followed for completion of Stage 1 of the website for BG Unified Solutions, an IT company based in Australia. The project documentation provides a detailed description of :

- The phases in which this stage of the website was completed.
- The advantages and use of the technology stack implemented.
- Goals set up to be achieved at the end of the stage.
- Prerequisite skills required for the team members to be a part of this team.
- Research done for the target audience and right deliverability.
- Parameters on which testing is dependent.
- The deployment process and the technologies used.
- Plan for the maintenance and enhancement of the website.
